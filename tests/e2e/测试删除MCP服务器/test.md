## 前置条件
需要先有一个已存在的 MCP 服务器。如果没有，先通过"添加MCP服务"功能添加一个测试服务器（名称为 "e2e-delete-test"，命令为 "echo"）。

## 测试步骤
1. 打开页面 http://localhost:9999/dashboard
2. 等待页面加载完成
3. 找到 MCP 服务器列表中的测试服务器 "e2e-delete-test"
4. 点击该服务器行的"卸载"按钮（红色文字，带有垃圾桶图标）
5. 截图（确认对话框打开状态）
6. 验证确认对话框标题包含"确定要删除这个(e2e-delete-test)MCP服务吗？"
7. 验证确认对话框描述包含"删除后，对应的工具列表也会移除。"
8. 点击"取消"按钮关闭对话框
9. 截图（取消删除后的状态）
10. 验证服务器仍然存在
11. 再次点击"卸载"按钮
12. 点击"确定"按钮确认删除
13. 等待操作完成
14. 截图（删除后的状态）
15. 验证 toast 提示包含"已删除"信息
16. 验证服务器已从列表中移除
