## 测试步骤
1. 打开页面 http://localhost:9999/dashboard
2. 等待页面加载完成
3. 找到并点击"添加 MCP 服务"按钮（带有 + 图标的按钮，title 为"添加MCP服务"）
4. 截图（对话框打开状态）
5. 验证对话框标题为"添加MCP服务"
6. 验证对话框包含"表单模式"和"高级模式"两个 tab
7. 在表单模式下：
   - 在"名称"输入框中输入测试名称 "e2e-test-server"
   - 在"命令"输入框中输入 "npx"
   - 截图（填写表单后的状态）
8. 点击"保存配置"按钮
9. 等待操作完成
10. 截图（添加后的状态）
11. 验证 toast 提示包含"已添加 MCP 服务"信息
12. 清理：删除刚添加的测试服务器以恢复环境
