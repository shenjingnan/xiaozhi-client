## 前置条件
- Dashboard 页面已打开且加载完成
- 系统处于正常工作状态

## 测试步骤

### 第一部分：系统设置保存后的数据一致性验证
1. 打开页面 http://localhost:9999/dashboard
2. 等待页面加载完成
3. 截图（初始状态）
4. 找到并点击系统设置按钮（齿轮图标按钮）
5. 截图（系统设置对话框打开状态）
6. 验证对话框标题为"系统设置"
7. 修改"心跳间隔（毫秒）"输入框的值（记录修改前的值和修改后的值）
8. 点击"保存"按钮
9. 等待保存操作完成
10. 验证 toast 提示包含保存成功信息
11. 关闭系统设置对话框
12. 重新打开系统设置对话框
13. 验证"心跳间隔（毫秒）"输入框的值与刚才修改后的值一致
14. 将值恢复为原始值并保存
15. 截图（数据一致性验证通过）

### 第二部分：MCP 服务器添加后的全局数据一致性验证
16. 找到并点击"添加 MCP 服务"按钮
17. 在表单模式下填写测试服务器信息：
    - 名称：e2e-config-consistency-test
    - 命令：npx
18. 点击"保存配置"按钮
19. 等待操作完成
20. 验证 toast 提示包含"已添加 MCP 服务"
21. 截图（添加服务器后的 Dashboard 状态）
22. 验证新添加的服务器 "e2e-config-consistency-test" 出现在 MCP 工具表格的服务器列表中
23. 刷新页面（F5 或重新导航到 /dashboard）
24. 等待页面加载完成
25. 验证刷新后服务器 "e2e-config-consistency-test" 仍然存在于列表中（确认配置已持久化）

### 清理步骤
26. 找到服务器 "e2e-config-consistency-test" 所在行
27. 点击"卸载"按钮
28. 在确认对话框中点击"确定删除"
29. 验证删除成功提示
30. 截图（清理完成状态）
