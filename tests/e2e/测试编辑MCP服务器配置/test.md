## 测试步骤
1. 打开页面 http://localhost:9999/dashboard
2. 等待页面加载完成
3. 找到 MCP 服务器列表中的任意一个服务器
4. 点击该服务器行的"配置"按钮（带有齿轮图标）
5. 截图（配置对话框打开状态）
6. 验证对话框标题为"配置 XXX MCP"（XXX 为服务器名称）
7. 验证对话框描述包含"点击保存后，需要重启服务才会生效。"
8. 验证包含"表单模式"和"高级模式 (JSON)"两个 tab
9. 验证表单模式中的输入框已加载现有配置数据
10. 截图（表单模式状态）
11. 点击"高级模式 (JSON)" tab
12. 截图（JSON 模式状态）
13. 验证 JSON 编辑器中显示当前配置的 JSON 内容
14. 点击"表单模式" tab 切换回表单视图
15. 截图（切换回表单模式的状态）
16. 关闭对话框（点击取消或 X 按钮）
