## 测试步骤
1. 打开页面 http://localhost:9999/dashboard
2. 等待页面加载完成
3. 找到并点击系统设置按钮（带有齿轮图标的按钮，位于导航栏右侧或状态卡片区域）
4. 截图（系统设置对话框打开状态）
5. 验证对话框标题为"系统设置"
6. 验证对话框描述为"配置平台认证和连接参数"
7. 验证包含以下配置项：
   - "魔搭社区 API Key" 输入框（带有"打开魔搭社区"按钮）
   - "扣子身份凭证" 输入框（带有"打开扣子平台"按钮）
   - "心跳间隔（毫秒）" 输入框
   - "心跳超时（毫秒）" 输入框
   - "重连间隔（毫秒）" 输入框
8. 验证对话框底部包含"取消"和"保存"按钮
9. 截图（完整设置界面）
10. 关闭对话框（点击取消按钮）
