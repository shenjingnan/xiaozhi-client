# 作为 MCP 服务使用

<Callout type="info">
xiaozhi-client 不仅可以连接小智服务端接入点，还可以作为普通 MCP 服务被其他客户端集成。
</Callout>

## 准备工作

首先，你需要先启动服务

```bash
xiaozhi start
```

<Callout type="info">服务启动后，你可以使用以下配置添加到你的MCP客户端中</Callout>

```json mcpServer
{
  "mcpServers": {
    "xiaozhi-client": {
      "type": "streamableHttp",
      "url": "http://localhost:9999/mcp"
    }
  }
}
```
