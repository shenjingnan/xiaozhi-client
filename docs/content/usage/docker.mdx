### 方式三：手动启动

<Steps>
## 创建挂载点目录
  ```bash
  mkdir -p ~/xiaozhi-client
  ```

## 拉取镜像并启动服务
    ```bash
    docker run -d \
    --name xiaozhi-client \
    -p 9999:9999 \
    -v ~/xiaozhi-client:/workspaces \
    --restart unless-stopped \
    shenjingnan/xiaozhi-client
    ```
</Steps>
